# Palworld Build Mode AirTest QA

## 프로젝트 개요
Palworld 건축모드 기능을 대상으로 테스트 케이스를 작성하고, AirTest 자동화 적용 가능성을 검토한 QA 포트폴리오입니다.

## 테스트 범위
- 건축모드 진입 / 종료
- 건축 탭 전환
- 건축물 프리뷰 확인
- 설치 가능 / 설치 불가 상태 확인
- 건축 입력, 회전, 스냅 정렬, 취소
- 해체 모드 진입 및 중단

## 자동화 분류 기준
- Candidate: 반복 실행 가치가 높고 이미지 인식 또는 UI 상태 판정이 가능한 케이스
- Manual: 지형, 카메라, 설치 위치, 시각적 판단 등 사람의 확인이 더 적합한 케이스
- Automated: 현재 구현 완료된 항목 없음

## 현재 진행 상태
- 테스트 케이스 작성 완료
- 자동화 후보 분류 완료
- AirTest 자동화 설계 진행 중
- 자동화 스크립트는 추후 구현 예정

## 사용 예정 도구
- AirTest
- Python
- 이미지 기반 UI 판정
- Excel / Google Sheets 기반 테스트 케이스 관리

## 주의
본 저장소는 자동화 구현 완료 프로젝트가 아니라, 게임 QA 자동화 설계와 자동화 후보 선정 과정을 정리한 포트폴리오입니다.
