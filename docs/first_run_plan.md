# 1차 자동화 실행 계획

## 1. 먼저 돌릴 구간

처음에는 전체 100개 Candidate를 한 번에 실행하지 말고 아래 순서로 나눠서 실행하세요.

1. Mode 케이스
   - PW-BLD-001 ~ PW-BLD-004
2. Tab 케이스
   - PW-BLD-005 ~ PW-BLD-014
3. Preview 케이스
   - 각 건축물의 `프리뷰` 케이스
4. 조건부 케이스
   - 벽 없음 차단
   - 벽면 설치 가능
   - 토대 없음 차단
   - 토대 설치 후 가능
5. 실제 설치/해체 케이스

## 2. 이미지 캡처 우선순위

1. 건축 모드 창 이미지
2. 각 탭 이미지
3. 각 탭 대표 건축물 이미지
4. 건축 프리뷰 UI 이미지
5. 프리뷰 조작 안내 이미지
6. 설치 완료 건축물 이미지
7. 벽/토대 fixture 관련 이미지

## 3. 성공 기준

- 같은 케이스를 3회 반복 실행했을 때 모두 통과
- 실패 시 AirTest 로그와 스크린샷으로 원인 확인 가능
- 좌표 의존성이 너무 크면 Manual 유지
